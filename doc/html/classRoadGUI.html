Detailed Description

The RoadGUI class. Class holds info about look of roads in GUI.

Class holds information necessary to draw road object on screen. It implements virtual function draw. It additionally has function drawSidewalk, so sidewalks can be drawn below all roads. Without caling this function, sidewalks won't appear. It also holds all information about how roads and sidewalks are supposed to look.

Detailed Description

Represents segment line. Stores straight line equation: (y - ya) * (xb - xa) - (yb - ya) * (x - xa) = 0 y * (xb - xa) - x * (yb - ya) = ya * (xb - xa) - xa * (yb - ya) and limits of segment line (x and y limits)

Detailed Description

The CameraGUI class. Class holds info about look of camera in GUI.

Class holds information necessary to draw camera in GUI. It implements virtual function from base class, so it wil be drawn poperly. It has also function that allows io move it and differ its range of sight. Function holds all information about the look of the camera on screen.
